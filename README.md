# 🚀 Financial Assistant Telegram Bot (Moliyaviy Yordamchi Bot)

A feature-rich Telegram bot built to help users manage their daily tasks and track personal finances efficiently. The bot provides an intuitive interface using custom reply keyboards to log expenses, check reports, and export records.

## ✨ Features

* **📋 Vazifalar (Tasks):** Manage and monitor your daily tasks or financial goals.
* **📓 Kundalik (Daily Log):** Quickly record day-to-day income and expenses.
* **📊 Hisobot (Reports):** Generate visual or text-based summaries of your financial data.
* **📥 Excel Yuklash (Export to Excel):** Download all your logged data directly into an Excel spreadsheet for advanced tracking.

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Framework:** `python-telegram-bot` or `aiogram` (Async Telegram Bot API)
* **Database:** PostgreSQL / SQLite (for storing user records and logs)
* **Data Export:** `pandas` / `openpyxl` (for Excel generation)
